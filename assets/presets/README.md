# Preset Images

This folder contains preset images that users can choose from when configuring their marketing materials.

## Folder Structure

- `flyer-back/` - Preset background images for the back of flyers (before/after photos, work samples, etc.)

## Image Guidelines

### Flyer Back Images
- **Recommended size**: 800x400px or similar landscape ratio
- **Format**: JPG or PNG
- **Content ideas**: Before/after comparisons, team photos, van shots, work in progress

## Adding New Presets

1. Add your images to the appropriate folder
2. Update the preset arrays in `app/materials.tsx` to include the new images
